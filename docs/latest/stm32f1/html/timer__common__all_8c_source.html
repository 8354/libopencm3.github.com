19 @section tim_api_ex Basic TIMER handling API.
20 
21 Enable the timer clock first. The timer mode sets the clock division ratio, the
22 count alignment (edge or centred) and count direction. Finally enable the
23 timer.
24 
25 The timer output compare block produces a signal that can be configured for
26 output to a pin or passed to other peripherals for use as a trigger. In all
27 cases the output compare mode must be set to define how the output responds to
28 a compare match, and the output must be enabled. If output to a pin is
29 required, enable the appropriate GPIO clock and set the pin to alternate output
30 mode.
31

34  rcc_peripheral_enable_clock(&RCC_APB2ENR, RCC_APB2ENR_TIM2EN);

36  timer_set_mode(TIM2, TIM_CR1_CKD_CK_INT_MUL_2,

40  timer_enable_counter(TIM2);
41 @endcode

222 
223 Returns true if the specified interrupt flag (UIF, TIF or CCxIF, with BIF or
224 COMIF for advanced timers) was set and the interrupt was enabled. If the
225 specified flag is not an interrupt flag, the function returns false.

237 {

285 /** @brief Set the Timer Mode.
286 
287 The modes are:
288 
289 @li Clock divider ratio (to form the sampling clock for the input filters,
290 and the dead-time clock in the advanced timers 1 and 8)
291 @li Edge/centre alignment
292 @li Count direction
293 
294 The alignment and count direction are effective only for timers 1 to 5 and 8
295 while the clock divider ratio is effective for all timers except 6,7
296 The remaining timers are limited hardware timers which do not support these mode
297 settings.
298 
299 @note: When center alignment mode is selected, count direction is controlled by
300 hardware and cannot be written. The count direction setting has no effect
301 in this case.
302 
303 @param[in] timer_peripheral Unsigned int32. Timer register address base @ref
304 tim_reg_base (TIM1, TIM2 ... TIM5, TIM8)
305 @param[in] clock_div Unsigned int32. Clock Divider Ratio in bits 8,9: @ref
306 tim_x_cr1_cdr
307 @param[in] alignment Unsigned int32. Alignment bits in 5,6: @ref tim_x_cr1_cms
308 @param[in] direction Unsigned int32. Count direction in bit 4,: @ref
309 tim_x_cr1_dir
310 */
311 
312 void timer_set_mode(uint32_t timer_peripheral, uint32_t clock_div,
313  uint32_t alignment, uint32_t direction)
314 {
315  uint32_t cr1;
316 
317  cr1 = TIM_CR1(timer_peripheral);
318 
319  cr1 &= ~(TIM_CR1_CKD_CK_INT_MASK | TIM_CR1_CMS_MASK | TIM_CR1_DIR_DOWN);
320 
321  cr1 |= clock_div | alignment | direction;
322 
323  TIM_CR1(timer_peripheral) = cr1;
324 }
